Hva er Node.js?

Node.JS, ofte referert til som Node, er en JavaScript -runtime -motor som lar deg kjøre JS -kode utenfor en nettleser. Node er basert på Googles V8 JavaScript -motor, den samme som ble brukt til behandling i Chrome. Det brukes ofte til å bygge applikasjoner, arbeide med moderne webapplikasjonsrammer og lage skript på serversiden med JS. Hva kan man gjøre med Node.js?

JavaScript -kode fungerer litt annerledes når den kjøres i Node. Først og fremst er det ingen DOM (HTML lastet fra en webside), så du kan ikke få tilgang til HTML -elementer eller bruke et bibliotek som jQuery med stor effekt (selv om det er alternativer). Men du har tilgang til systemressurser og kan lese og skrive filer, lage nettverkstilkoblinger og til og med betjene programmer.

Hvilke apper bruker node?

Node brukes også til å lage skrivebordsprogrammer, ved bruk av et rammeverk som kalles Electron. Elektron driver Slack, Discord, Visual Studio Code og Skype. I hovedsak kjører den en lite versjon av Chrome designet for skrivebordet, og selve appen er faktisk bare en nettapp. Som et resultat er Electron -apper veldig sultne på RAM og vil fungere litt dårligere enn sine opprinnelige kolleger, men denne handelen gjøres ofte for å være konsistent på tvers av alle operativsystemer og på nettet.
